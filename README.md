## 📚 项目简介

本仓库是华中科技大学计算机科学与技术专业的课程学习资料整理与备份，涵盖**大一至大三**共 **37+ 门课程**的学习资料、实验代码、作业、课程项目等内容。

旨在为学弟学妹们提供学习参考，帮助大家更好地完成课程学习。欢迎 Star ⭐ 和分享！

> ⚠️ **注意事项**
> - 本仓库仅供学习交流使用，请勿直接抄袭作业
> - 部分内容会在课程结课后陆续更新
> - 请尊重知识产权，合理使用资料

---

## ✨ 特色功能

- 📖 **完整的课程体系**：覆盖计算机专业核心课程
- 💻 **实践代码**：包含实验代码、头歌平台代码、课程设计项目
- 📝 **学习笔记**：部分课程包含学习笔记和总结
- 🎯 **项目实战**：真实的课程项目案例（如软件工程项目）
- 📦 **便捷下载**：支持单个文件夹下载

## 🚀 使用指南

### 📥 下载整个仓库

```bash
git clone https://github.com/Ilosyi/Hust-CS-Learning-Library.git
```

### 📦 下载特定文件夹

由于仓库内容较多，如果只需要某个课程的资料，可以使用以下方法：

**方法一：使用 DownGit** （推荐）

1. 访问 [DownGit](https://minhaskamal.github.io/DownGit/)
2. 复制你需要的文件夹 URL（例如：`https://github.com/Ilosyi/Hust-CS-Learning-Library/tree/main/IB 数据结构`）
3. 粘贴到 DownGit 输入框，点击下载

**方法二：使用 GitHub 命令行工具 gh**

```bash
# 安装 gh (如果未安装)
# 然后使用 sparse-checkout
git clone --depth 1 --filter=blob:none --sparse https://github.com/Ilosyi/Hust-CS-Learning-Library.git
cd Hust-CS-Learning-Library
git sparse-checkout set "IB 数据结构"
```

**方法三：浏览器扩展**

- Chrome/Edge: [GitZip](https://chrome.google.com/webstore/detail/gitzip-for-github/ffabmkklhbepgcgfonabamgnfafbdlkn)

## 🤝 贡献指南

欢迎大家贡献自己的学习资料！

### 贡献方式

1. Fork 本仓库
2. 创建新的分支 (`git checkout -b feature/add-course`)
3. 添加或修改内容
4. 提交更改 (`git commit -m 'Add: 添加某课程资料'`)
5. 推送到分支 (`git push origin feature/add-course`)
6. 提交 Pull Request

### 文件命名规范

- 课程文件夹：`学期 课程名称`（如：`IB 数据结构`）
- 代码文件：使用有意义的文件名，建议包含题目或功能描述
- 文档文件：使用中文命名，清晰描述内容

### 内容建议

- ✅ 确保上传的代码可以正常运行
- ✅ 添加必要的注释和说明
- ✅ 保护隐私，不上传包含个人敏感信息的文件
